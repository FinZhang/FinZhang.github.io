捷号決戦！邀撃、レイテ沖海戦(后篇)

2018-07-26
#Event

2018年冬季期间限定海域作战记录

E1 作戦开始 前路哨戒を厳とせよ！

警戒条。照例开门反潜。I,F,BOSS点为水上水下混合部队，进N和boss有索敌需要。纯水雷6.79可进，带BB索敌要上11。

开路

四小船出门走A点亮B点。

1CL4DD1BB/2CL4DD出门手动走F和G打出F和N点A。带BB会拐J但问题不大。

攻略

上述配置A上去就完了。

龙田 夕张 文月 皋月 占守 国后 索敌 8.1

龙田 甘古特 文月 皋月 占守 国后 索敌 15.6

E2 威风堂々 出撃！栗田舰队

西村条/栗田条。凑数的图……

开路

西村编满出门打出S,X,Y点A，空均144.

扶桑 山城 最上 时雨 满朝 朝云 山云

栗田联合出门到达T点

攻略

铃谷熊野罚站（当时还没有改成CVL，制空很尴尬），空均进场。

利根 筑摩 金刚 榛名 铃谷 熊野

能代 早霜 羽黑 照月 北上 大井

2.18完成

E3 小沢机动部队 全力出撃!

小泽条。我tm被tsu爷社保……

开路只能少打一个点，boss太鸡不需要。史实船大于等于8

全程550制空出门，陆航121121集中boss

瑞鹤 翔鹤 千代田 千岁 伊势 日向

大淀 初月 秋月 木曾 高雄 爱宕

斩杀时高雄换大井，陆航211121集中boss

除了耗铝没什么好说的……主任已经不适合这个版本了

E4 サマール沖 その先へ――

西村条/栗田条。难度骤升，为了给后段留点SAN切了乙，秒变脚打。

开路

西村出门打出N点A

扶桑 山城 最上 时雨 满朝 朝云 山云

攻略

武藏 榛名 长门 金刚 铃谷 熊野

能代 长波 雪风 摩耶 北上 大井

出门制空503，陆航40压苍蝇点04集中门神

双支援（切乙后撤除）

2.21完成

E5 多号作戦改

志摩条。借助单舰退避运输阶段会出现3船进王靠陆航混A的奇观……

斩杀有点看脸，但总体难度不大。

运输

制空116，索敌11.2可进

特斯拉 那智 足柄 阿武隈 鬼怒 霞 潮

陆航04boss40集中F，斩杀1304集中boss保A

完成运输后换成全水战打出D点空优

攻坚

三隈 足柄 阿武隈 凉月 初霜 潮 霞

单水战赌空劣触发夜侦

磨血03+东海集中门神13集中boss，斩杀1313全集中boss

2.23完成

E6 追撃！第二遊撃部隊

志摩条/栗田条。一图3吃，追击战不带空母脸接空丧空袭，这届大本营脑子进水.jpg

P1 志摩

三隈 俾斯麦 意大利 足柄 阿武隈 凉月 霞

5水战，陆航13去O 13集中boss. H点是反潜战。

P2 栗田

利根 金刚 榛名 筑摩 北上 长波 雪风

4水战出门，斩杀时1水战挤一个二连出来。斩杀输出压力非常大，陆航全boss.

完成后打出K点A和I点S，三次守家空优开出P3 boss点。

P3 志摩

足柄 衣阿华 俾斯麦 三隈 凉月 霞 北上

P3难度比P2低一点，然后进入史诗地狱难度捞船，劝退和吃桶率可媲美当年捞矶风。

所幸皇天不负有心人。同志，伏特加 !( 吨吨吨.jpg )

2.25完成 (25一整天都在捞船)

E7 暁の水平線に勝利を――

LET’S DO THIS.

P1 战舰水鬼改

开点

机动出门开点，打出Q点A加两次守家空优

翔鹤 瑞鹤 大凤 Saratoga 意大利 俾斯麦

神通 秋月 朝潮 摩耶 北上 大井

攻坚

高速水打，一队东海压鱼点，一队集中boss

磨血时可以单cv或者双cvl，没啥区别，总共就两段血（虽然我一次都没打沉过）

武藏 衣阿华 金刚 榛名 瑞鹤/熊野 利根/铃谷

阿武隈-CI 摩耶-二连 长波-二连 北上-二连 雪风-CI 瑞凤-CI

斩杀时全员倍卡上速吸，二队位置太过宝贵所以放一队尾插上损管，剩下的就是展现人类意志力了。

武藏 大和 金刚 榛名 瑞鹤 速吸

阿武隈-CI 摩耶-CI 妙高-CI 绫波-CI 夕立-CI 瑞凤-CI

理事长改在没有友军舰队支援前攻略难度极高，极高的耐久需要本队在昼战击沉僚舰并压制血线，才能保证夜战暴击的CI能打到理事长改身上。但我们终究成功了。

顶配夜战套瑞凤CI打出412斩杀，不枉我火线改出来的F6F-5N。而且也算给萌新时的自己报仇了。

虽然没法切实证明是零灯斩杀的=。=不过我问心无愧。

P2 深海鹤栖姬

开路

高速水打出门打出H点反潜S和L点S，直接沿用P1攻略阵容就可以。

低速机动打出I点空优、C点空优和G点S，用P1开路换个低速船的阵容。

5BB混速水打打出Q点A，这里舰队里需要有四种贴条的船。

高速水打再次出门打出W点A胜，完成最终出击准备。

攻坚

有了倍卡之后和P1的叹息之墙比起来要简单不少，晚上还有友军舰队压血线。

磨血

榛名 武藏 翔鹤 瑞鹤 大凤 Saratoga

阿武隈 北上 朝潮 摩耶 夕立 瑞凤

斩杀

榛名 武藏 翔鹤 瑞鹤 大凤 加贺

阿武隈 摩耶 妙高 雪风 瑞凤 霞

霞夜战一发CI暴击overkill.

在拂晓的水平线上，镌下胜利!

眼眶热着听完月夜海后我决定（向爽哥学习）单方面宣布田中为2018年最佳游戏制作人
