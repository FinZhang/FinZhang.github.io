捷号決戦！邀撃、レイテ沖海戦(前篇)

2017-11-28
#Event

2017年秋季期间限定海域作战记录

E1 第二遊撃部隊、抜錨！

这回从E1就有解密，一段开路二段开boss点。

开路

去一趟F再去一趟K自动打开JM以及后段路线，此段编入秋津丸提高制空。

潮 朝霜 Pola 秋津丸 千岁 初霜

双开幕反潜，初霜对空CI，制空246，陆航40集中E点。

开路后再去一趟E打出一个空优，然后点亮P点。随后再次进击到达门神O点时在进入战斗前BOSS点就会出现。

本阶段阵容不需要变化，若E打不出空优建议洗脸，敌方有高低制空配置。

攻略

此段注意需要三志摩舰队成员带路，具体成员后附。

注意索敌，25.19可进Boss

潮 朝霜 足柄 那智 鬼怒 初霜

道中反潜支援，决战炮击支援，陆航04集中boss点或者洗门神提高通过率。

11.19完成

E2 捷一号作戦、発動準備

E2可以第三舰队组成游击舰队出击，7条船好处就不多说了。

开路

老样子要开路，先做C点空优+A胜以上和J点胜利。可以全带反潜装备。

潮 那智 意大利 霞 朝霜 千岁 秋津丸

然后更换秋津丸为CVL，拐下做D点空优+A胜以上。注意配合40陆航，水战水爆带好。

攻略

此P为运输作战，CLDD优先选择带路船中有大发的。最短路线对配置要求太苛刻，选择次短路线上两个高速BB。

路上多打一个水雷点问题不大，劝退主要是双タ姐点。

那智 意大利 龙骧 鬼怒 罗马 潮 霞

道中炮击支援，追求boss点S的带好决战支援。个人6大发3桶，A58 S83。

11.20完成

E3 捷一号作戦、作戦発動！

一图三吃，这根本不是中型活动(:зゝ∠)

P1

此段作战又叫防空姬无惨。有带路，但是完全可以无视带路黄金舰队走最长路骑防空姬脸。

武藏 大和 衣阿华 厌战 飞鹰 隼鹰

夕立 照月 神通 朝潮 大井 北上

boss点火力基本溢出，带四先制反潜压制鱼点，一个对空CI防空袭爆炸。

P2

换成运输弱鸡舰队后道中就比较凶险了，带好道中支援，一队高反潜陆航+一到两个先制保证鱼点通过率。

注意有索敌要求，不过一般没问题。

进去打A比较容易，S需要一些脸（遇到有雷爷的阵容就不要想了）。

熊野 铃谷 如月 江风 荒潮 睦月

由良 欧根 鸟海 大潮 响 朝潮

带够大发A也有三位数运输量，很快就能解决。

运输完后陆航两队40守家，需要一次空袭白字才能开始P3的航线图。在制空状态不改变的情况下尽量上陆战局战，有额外击坠数。脸黑可能要做很多次（比如我）

P3

打主任，注意需要足够小泽舰队的船（6艘）才能走最短路线。

深海侧制空非常高，磨血590+出门，陆航40门神boss各一波，13集中boss；

斩杀680+出门，双对空CI，陆航2222集中boss，或者放弃陆航输出40洗boss制空，这样主队出门制空要求能压低到500以下。新的战爆联合炮击战伤害很高。主任甲薄，夜战不需要太多斩杀火力。

Zara 日向 瑞鹤 大凤 千代田 千岁

欧根 神通 羽黑 北上 秋月 初月

11.21完成

E4 あの海峡の先へ――

“现在，我们都是西村提督了。”

个人比较满意的图，史实解密和贴条策略彰显了田中君的游戏策划能力，演出表现也极佳。只是道中配置还是挺多shi的，让这张图没法拿到满分。

P1

开路第一阶段，任务表：U点A，Y点A，X点亮，MNV空优+A胜以上，一次守家空袭无伤。

1.7船组成志摩舰队，要求全队贴上志摩舰队的条子，从3号出发点出发。

路线QTVXY，做掉Y点的A，V点空优和A，X点亮

足柄 那智 潮 鬼怒 霞 阿武隈 龙骧

注意要走X需要换掉龙骧，换成秋津丸可能能一次做完，我为了保险上了CVL分两次做空优和点亮A。

2.7船组成西村舰队，注意至少6条西村船（甲难度），从1号出发点出发。、

路线EIMN，做MN的空优和A，路上只有一个反潜，所以能带飞机的船全部罚站保证两点空优。

扶桑 山城 最上 朝云 满朝 时雨 瑞凤

全罚站制空507，注意可以配合陆航。

3.组成联合舰队，栗田船+铃木船大于等于7，一队BB+CV系小于等于5绕开辛弃疾。

从2号出发点出发，做U点A胜。

金刚 榛名 衣阿华 那智 千岁 千代田

矢矧 羽黑 妙高 雪风 野分 朝潮

4.组个7ss（贴上西村条子）出门做空袭白字，三队40陆航守家。

路上没有能反潜的点，按住进击不要停，遇到空袭就回家。脸黑可能要多做几次（比如我）

做完后出Z1~Z9这9个点。

P2

开路第二阶段，任务表：守家空袭无伤一次，Z9点S，Z8点A，Z1Z2空优+A，Z6A

1.先和P1一样7ss出击做一次白字

2.7船西村舰队，6西村+1DD提高索敌做Z9点S，降低索敌做Z8A（可以故意退避一个来压索敌）

扶桑 山城 最上 朝云 满朝 时雨 绫波

陆航随意分配，推荐压制IMNS四点。

3.组成联合舰队，带路要求和P1一样。先带罚站双CVL做掉Z1Z2的空优和A

注意带武大长的话要提速

千岁 榛名 武藏 大和 铃谷 千代田

矢矧 鸟海 夕立 雪风 北上 大井

4040压两个空袭点，出门630。做完后换成单罚站CV做Z6S

这个S比较难打，建议武大提速二队上最强阵容。

金刚 榛名 武藏 大和 摩耶 Sara

神通 鸟海 夕立 雪风 北上 大井

打完后出现ZZ1~ZZ3航路，开始正式攻略。

P3

西村七剑满编，从2号出发点出发。DD双炮见张，最上三水战司令部旗舰，扶桑山城双炮彻甲弹侦察机，注意索敌高一些否则沟Z7。

扶桑 山城 最上 满朝 时雨 朝云 山云

道中注意S点退避一条DD会沟进R。陆航一队13集中S，一队04集中boss。磨血时陆航比较灵活，按自己情况分配。道中一律警戒阵。双支援需要。也可以考虑boss点也警戒阵靠擦伤磨血，此时不需要boss支援。

斩杀时boss制空提高，建议下最上换成一个SSV进行攻略。扶桑山城侦察机换成水爆，SSV双水战出门制空大约109。陆航2213集中boss压低制空，水爆不被打空可以拿下空优。

扶桑 山城 最上 满朝 时雨 朝云 伊13

可能索敌会不大够，有几个DD需要下见张换成电探。斩杀建议全队闪。

P4

打得过P3就打得过P4，难度是下降的。

斩杀依旧推荐SSV当坦。

boss防空过高，陆航跳海效果不佳，可以压制道中来提高通过率。

11.23完成

“越过苏里高，越过过去的梦魇，眼前将是崭新的未来。”
